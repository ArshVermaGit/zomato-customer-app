# Zomato By Arsh - App Store & Play Store Checklist

## iOS App Store

### App Information
- [ ] App Name: "Zomato By Arsh"
- [ ] Subtitle: "Order food from your favorite restaurants"
- [ ] Category: Food & Drink
- [ ] Age Rating: 4+
- [ ] Primary Language: English

### Required Assets
- [ ] App Icon (1024x1024, PNG, no alpha channel)
- [ ] Screenshots:
  - [ ] iPhone 6.5" (1284x2778) - minimum 3
  - [ ] iPhone 5.5" (1242x2208) - minimum 3
  - [ ] iPad Pro 12.9" (2048x2732) - if supporting iPad
- [ ] App Preview Video (optional, 30 seconds max)

### Metadata
- [ ] Keywords (100 characters max)
- [ ] Description (4000 characters)
- [ ] What's New (for updates)
- [ ] Privacy Policy URL
- [ ] Support URL
- [ ] Marketing URL (optional)

### Build & Signing
- [ ] Apple Developer Account ($99/year)
- [ ] App ID registered in Developer Portal
- [ ] Certificates: Distribution certificate

- [ ] Build number incremented

### Review Preparation
- [ ] Demo account (test user credentials for Apple Review)
- [ ] Review notes explaining any special features
- [ ] All third-party SDKs properly attributed
